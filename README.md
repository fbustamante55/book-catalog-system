# Book Catalog System

A Java-based application for managing a library's book catalog, including search, add, update, and delete operations.

## Technologies
- Java 11+
- Object-Oriented Programming

## Features
- Add, edit, and remove books
- Search by title, author, or genre
- Display full catalog
- Console-based interface

## Getting Started
```bash
javac Main.java
java Main
```
